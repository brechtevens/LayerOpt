Permittivity Models

File: scripts/Permittivity-example.py

This example demonstrates how to:

  1. Use built-in permittivity models such as simple linear and quadratic models.

  2. Create a custom permittivity model.

  3. Understand the physical sign convention for complex permittivity (ε = ε’ + iε’’).

  4. Visualize the real (ε’) and imaginary (ε’’) parts of permittivity.

Key classes used:

  • LayerOpt.Permittivity.PermittivityModel

  • LayerOpt.Permittivity.SimpleLinearPermittivityModel

  • LayerOpt.Permittivity.QuadraticPermittivityModel

Example code:

Example - Using and extending PermittivityModel classes
  1"""
  2Example: Using and extending PermittivityModel classes
  3
  4This script demonstrates:
  51. How to use the built-in permittivity models.
  62. How to create a custom permittivity model.
  7
  8Note:
  9-----
 10In this framework, **the imaginary part of permittivity (ε″) is negative
 11for a lossy material**. This follows the convention where the time-harmonic
 12field varies as exp(+iωt). With this definition, a negative ε″ corresponds
 13to energy dissipation (loss), while a positive ε″ would imply amplification.
 14"""
 15
 16import numpy as np
 17import matplotlib.pyplot as plt
 18from numba import njit
 19from LayerOpt.Permittivity import (
 20    SimpleLinearPermittivityModel,
 21    SimpleQuadraticPermittivityModel,
 22    PermittivityModel
 23)
 24
 25# ---------------------------------------------------------------------
 26# 1. Using a built-in permittivity model
 27# ---------------------------------------------------------------------
 28# Simple linear relation: ε' = p0 + p1*x, ε'' = p2 + p3*x
 29simple_model = SimpleLinearPermittivityModel(params=np.array([2.0, 0.1, -0.3, -0.2]))
 30
 31# Example input: material composition (wt%) and frequency (GHz)
 32x = np.linspace(0, 10, 100)  # finer grid for smooth plots
 33f = 10  # GHz
 34
 35eps_simple = simple_model.evaluate_permittivity(x, f)
 36
 37# ---------------------------------------------------------------------
 38# 2. Using a frequency-dependent quadratic model
 39# ---------------------------------------------------------------------
 40quad_model = SimpleQuadraticPermittivityModel(params=np.array([2.0, 0.1, 0.05, -0.3, -0.2, -0.05]))
 41eps_quad = quad_model.evaluate_permittivity(x, f)
 42
 43# ---------------------------------------------------------------------
 44# 3. Defining a custom permittivity model
 45# ---------------------------------------------------------------------
 46class SqrtPermittivityModel(PermittivityModel):
 47    """
 48        Example of a user-defined permittivity model.
 49
 50        Convention reminder:
 51        --------------------
 52        ε = ε' + iε″, with ε″ < 0 for lossy materials.
 53
 54        Always use
 55        
 56        @staticmethod
 57        @njit(cache=True)
 58        
 59        to ensure compatibility with
 60        Numba-accelerated calculations in MultilayerStructure.
 61    """
 62
 63    def get_default_params(self):
 64        # Example parameters: ε' = a + b*x + c*f, ε'' = d + e*x + g*f
 65        return np.array([2.0, 0.5, 0.001, -0.3, -1, -0.001])
 66
 67    def get_default_bounds(self):
 68        # Positive bounds for ε' coefficients, negative for ε'' coefficients
 69        return [*[(1e-6, np.inf)]*3, *[(-np.inf, -1e-6)]*3]
 70
 71    @staticmethod
 72    @njit(cache=True)
 73    def _compute_real_permittivity(x, f, p, p_extra):
 74        return p[0] + p[1]*np.sqrt(x) + p[2]*f
 75
 76    @staticmethod
 77    @njit(cache=True)
 78    def _compute_imaginary_permittivity(x, f, p, p_extra):
 79        return p[3] + p[4]*np.sqrt(x) + p[5]*f
 80
 81
 82# Instantiate and evaluate
 83custom_model = SqrtPermittivityModel()
 84eps_custom = custom_model.evaluate_permittivity(x, f)
 85
 86# ---------------------------------------------------------------------
 87# 4. Plotting the permittivity
 88# ---------------------------------------------------------------------
 89plt.figure(figsize=(10, 6))
 90
 91plt.subplot(2, 1, 1)
 92# Plot real parts ε'
 93plt.plot(x, eps_simple.real, label="Linear model")
 94plt.plot(x, eps_quad.real, label="Quadratic model")
 95plt.plot(x, eps_custom.real, label="Custom Sqrt model")
 96plt.ylabel("ε'")
 97plt.legend()
 98plt.grid(True)
 99
100plt.subplot(2, 1, 2)
101# Plot imaginary parts ε''
102plt.plot(x, eps_simple.imag, label="Linear model")
103plt.plot(x, eps_quad.imag, label="Quadratic model")
104plt.plot(x, eps_custom.imag, label="Custom Sqrt model")
105
106plt.xlabel("Material composition x (wt%)")
107plt.ylabel("ε''")
108plt.legend()
109plt.grid(True)
110plt.show()

Note

Numba JIT compilation of _compute_real_permittivity and _compute_imaginary_permittivity` is crucial!

If you remove @njit from _compute_real_permittivity or _compute_imaginary_permittivity, your model will still work for direct evaluation, but will fail when used inside LayerOpt.core.MultilayerStructure, because that class uses njit-compiled loops for performance.

In short:

  • Use ``@njit`` for permittivity models that will be simulated.

  • Skip ``@njit`` only for pure testing or debugging.

MultilayerStructure

File: scripts/MultilayerStructure-example.py

This example demonstrates how to:

  1. Define a multilayer dielectric structure.

  2. Use a simple linear permittivity model.

  3. Compute and visualize reflectance (R), transmittance (T), absorption (A), absorption-to-reflectance ratio (A/R), and shielding effectiveness (SET).

Key classes used:

  • LayerOpt.core.MultilayerStructure

  • LayerOpt.Permittivity.SimpleLinearPermittivityModel

Example code:

Example - Using the MultilayerStructure class
 1"""
 2Example: Using the MultilayerStructure class
 3
 4This script demonstrates how to:
 51. Define a multilayer dielectric structure.
 62. Use a simple linear permittivity model.
 73. Compute and visualize reflectance (R), transmittance (T), absorption (A),
 8   absorption-to-reflectance ratio (A/R), and shielding effectiveness (SET).
 9"""
10
11import numpy as np
12import matplotlib.pyplot as plt
13from LayerOpt.core import MultilayerStructure
14from LayerOpt.Permittivity import SimpleLinearPermittivityModel
15
16# ---------------------------------------------------------------------
17# 1. Define the multilayer structure
18# ---------------------------------------------------------------------
19# Five layers, each 5 mm thick, free-space impedance on the output side
20structure = MultilayerStructure(t=5e-3, n_lay=5, impedance_out=1)
21
22print(f"Total thickness: {structure.total_t:.3e} m")
23
24# ---------------------------------------------------------------------
25# 2. Define the permittivity model
26# ---------------------------------------------------------------------
27# Simple linear model: ε' = p0 + p1*wt, ε'' = p2 + p3*wt
28eps_model = SimpleLinearPermittivityModel(params=[2, 0.05, -0.25, -0.05])
29
30# Material composition per layer (weight %)
31x = np.array([0, 2, 4, 6, 8])
32
33# Frequency vector in GHz
34freq_vec = np.linspace(1, 20, 200)
35
36# ---------------------------------------------------------------------
37# 3. Compute desired electromagnetic properties
38# ---------------------------------------------------------------------
39# Electromagnetic properties for all frequencies in freq_vec
40E = structure.get_waves(x, freq_vec, eps_model)
41S11, S22 = structure.get_S_parameters(x, freq_vec, eps_model)
42R, T, A = structure.get_wave_properties(x, freq_vec, eps_model)
43AR, SET = structure.get_AR_SET(x, freq_vec, eps_model)
44
45# Worst-case (minimum) A/R and shielding effectiveness (SET) over all frequencies in freq_vec
46min_AR, min_SET = structure.get_worst_case_AR_SET(x, freq_vec, eps_model)
47
48print(f"Minimum A/R:  {min_AR:.3f}")
49print(f"Minimum SET:  {min_SET:.2f} dB")
50
51# ---------------------------------------------------------------------
52# 4. Plots included in package
53# ---------------------------------------------------------------------
54
55structure.visualize_behavior(x, freq_vec, eps_model)
56
57# ---------------------------------------------------------------------
58# 4. Other figures
59# ---------------------------------------------------------------------
60
61plt.figure(figsize=(8, 6))
62
63plt.subplot(3, 1, 1)
64plt.plot(freq_vec, R, marker='.', linestyle="", label="Reflectance (R)")
65plt.plot(freq_vec, T, marker='.', linestyle="", label="Transmittance (T)")
66plt.plot(freq_vec, A, marker='.', linestyle="", label="Absorption (A)")
67plt.xlabel("Frequency [GHz]")
68plt.ylabel("Coefficient")
69plt.legend()
70plt.grid(True)
71
72plt.subplot(3, 1, 2)
73plt.plot(freq_vec, AR, marker='.', linestyle="")
74plt.xlabel("Frequency [GHz]")
75plt.ylabel("A/R")
76plt.grid(True)
77
78plt.subplot(3, 1, 3)
79plt.semilogy(freq_vec, SET, marker='.', linestyle="")
80plt.xlabel("Frequency [GHz]")
81plt.ylabel("Shielding Effectiveness [dB]")
82plt.grid(True)
83
84plt.tight_layout()
85plt.show()

Note that one can use any other Permittivity model in this example script, such as

  • LayerOpt.Permittivity.LinearPermittivityModel

  • LayerOpt.Permittivity.SimpleQuadraticPermittivityModel

Note

When running the code, you may see a warning such as:

NumbaPerformanceWarning: np.dot() is faster on contiguous arrays, called on (Array(complex128, 2, 'A', False, aligned=True), Array(complex128, 1, 'C', False, aligned=True))

This is a harmless performance warning from Numba and can be safely ignored. It does not affect correctness or numerical results.

Example - Using the ExperimentData class
 1"""
 2Example: Using ExperimentData class

 4This script demonstrates how to:
 5
 61. Load experimental multilayer measurement data.

 83. Plot A/R versus shielding effectiveness (SET) for different experiments.

10Note:
11-----

Learning Permittivity Models

File: scripts/MultilayerLearner-example.py

This example demonstrates how to:

  1. Learn permittivity model parameters from experimental data using LayerOpt.core.MultilayerLearner with a user-defined cost function.

  2. Visualize and compare predicted versus measured transmittance (T), reflectance (R), and absorption (A).

Key classes used:

  • LayerOpt.core.MultilayerLearner

  • LayerOpt.Permittivity.SimpleQuadraticPermittivityModel

  • LayerOpt.Experiment.ExperimentData

Example code:

Example - Learning permittivity model parameters with MultilayerLearner
  1"""
  2Example: Learning permittivity model parameters with MultilayerLearner
  3
  4This script demonstrates how to:
  5
  61. Learn permittivity model parameters from experimental data using MultilayerLearner with a user-defined cost function.
  72. Visualize and compare predicted versus measured transmittance (T), reflectance (R), and absorption (A).
  8"""
  9
 10import os
 11import time
 12import matplotlib.pyplot as plt
 13
 14from LayerOpt.Experiment import ExperimentData
 15from LayerOpt.Permittivity import SimpleQuadraticPermittivityModel
 16from LayerOpt.core import MultilayerLearner
 17
 18# ---------------------------------------------------------------------
 19# 1. Load experimental data
 20# ---------------------------------------------------------------------
 21data_dir = os.path.join(os.path.dirname(__file__), '../data/')
 22exp_data = ExperimentData(directory=data_dir, reverse=False)
 23experiment_names = list(exp_data.get_names())
 24print("Loaded experiments:", experiment_names)
 25
 26# ---------------------------------------------------------------------
 27# 2. Initialize a simple quadratic permittivity model
 28# ---------------------------------------------------------------------
 29eps_model = SimpleQuadraticPermittivityModel()
 30print("Default model parameters:", eps_model.get_default_params())
 31
 32# ---------------------------------------------------------------------
 33# 3. Create the learner
 34# ---------------------------------------------------------------------
 35learner = MultilayerLearner(measurements=exp_data, eps_model=eps_model)
 36
 37# ---------------------------------------------------------------------
 38# 4. Define a cost function and evaluate the initial cost
 39# ---------------------------------------------------------------------
 40def custom_cost(p):
 41    errors = 0
 42    for experiment_name in exp_data.get_names():
 43        wt = exp_data.get_material_parameters(experiment_name)
 44        freq_range = exp_data.get_X(experiment_name)
 45        eps_model.p = p
 46        R_pred, T_pred, A_pred = exp_data.structure[experiment_name].get_wave_properties(wt, freq_range, eps_model)
 47        R_meas = exp_data.get_R(experiment_name)
 48        # sum of squared errors only for T
 49        errors += ((R_pred - R_meas)**2).sum()/exp_data.n
 50    return errors
 51
 52cost_fun = custom_cost
 53p0 = eps_model.get_default_params()
 54initial_cost = cost_fun(p0)
 55print("Initial cost:", initial_cost)
 56
 57# ---------------------------------------------------------------------
 58# 5. Optimize permittivity parameters
 59# ---------------------------------------------------------------------
 60optimal_params, optimal_cost = learner.optimize(p0, cost_fun)
 61print("Optimized parameters:", optimal_params)
 62print("Optimized cost:", optimal_cost)
 63
 64# ---------------------------------------------------------------------
 65# 6. Compare measured vs predicted coefficients
 66# ---------------------------------------------------------------------
 67num_experiments = len(exp_data.get_names())
 68plt.figure(figsize=(6, 4 * num_experiments))
 69
 70for j, experiment_name in enumerate(exp_data.get_names()):
 71    plt.subplot(num_experiments, 1, j+1)
 72    wt = exp_data.get_material_parameters(experiment_name)
 73    freq_range = exp_data.get_X(experiment_name)
 74
 75    # Predicted coefficients
 76    R_pred, T_pred, A_pred = exp_data.structure[experiment_name].get_wave_properties(
 77        wt, freq_range, eps_model
 78    )
 79
 80    # Measured coefficients
 81    T_meas = exp_data.get_T(experiment_name)
 82    R_meas = exp_data.get_R(experiment_name)
 83    A_meas = exp_data.get_A(experiment_name)
 84
 85    # Plot T
 86    plt.plot(freq_range, T_meas, 'o', color='tab:blue', label='T measured')
 87    plt.plot(freq_range, T_pred, '-', color='tab:blue', label='T predicted')
 88
 89    # Plot R
 90    plt.plot(freq_range, R_meas, 'o', color='tab:orange', label='R measured')
 91    plt.plot(freq_range, R_pred, '-', color='tab:orange', label='R predicted')
 92
 93    # # Plot A
 94    plt.plot(freq_range, A_meas, 'o', color='tab:green', label='A measured')
 95    plt.plot(freq_range, A_pred, '-', color='tab:green', label='A predicted')
 96
 97    plt.xlabel("Frequency [GHz]")
 98    plt.ylabel("Coefficient")
 99    plt.title(f"{experiment_name}")
100    plt.grid(True)
101    plt.legend()
102
103plt.tight_layout(pad=3.0, w_pad=3.0, h_pad=3.0)
104plt.subplots_adjust(wspace=20)
105plt.show()
106
107
108

Note

In this example, a custom cost function is used that only penalizes errors in R (reflectance). This illustrates how the choice of cost function affects the predicted coefficients: R is accurately matched, but T and A are not. To optimize across all coefficients (T, R, A) and compare, re-run the script using cost_fun = learner.cost_coeffs().

Optimizing Multilayer Structures

File: scripts/MultilayerOptimizer-example.py

This example demonstrates how to:

  1. Optimize the material parameters of a multilayer dielectric slab structure with a given permittivity model. The goal is to minimize R/A (or equivalently to maximize A/R), while satisfying a shielding effectiveness (SE) constraint.

  2. Visualize the optimization results.

Key classes used:

  • LayerOpt.core.MultilayerOptimizer

  • LayerOpt.core.MultilayerStructure

  • LayerOpt.Permittivity.SimpleLinearPermittivityModel

Example code:

Example - Optimizing multilayer dielectric slab structures with MultilayerOptimizer
 1"""
 2Example: Optimizing multilayer dielectric slab structures with MultilayerOptimizer
 3
 4This script demonstrates how to:
 5
 61. Optimize the material parameters of a multilayer dielectric slab structure with a given permittivity model. The goal is to minimize R/A (or equivalently to maximize A/R), while satisfying a shielding effectiveness (SE) constraint.  
 72. Visualize the optimization results.
 8"""
 9
10import os
11import time
12import numpy as np
13import matplotlib.pyplot as plt
14
15from LayerOpt.Permittivity import SimpleLinearPermittivityModel
16from LayerOpt.core import MultilayerStructure, MultilayerOptimizer
17
18# ---------------------------------------------------------------------
19# 1. Define a simple multilayer structure
20# ---------------------------------------------------------------------
21# Example: 5 layers, each 5 mm thick, outgoing impedance = 1 (free space)
22n_layers = 5
23MS = MultilayerStructure(t=5e-3, n_lay=n_layers, impedance_out=1)
24
25# Initial guess for material parameters
26x0 = np.linspace(0.2, 0.8, n_layers)
27
28# ---------------------------------------------------------------------
29# 2. Define a permittivity model
30# ---------------------------------------------------------------------
31eps_model = SimpleLinearPermittivityModel()
32
33# ---------------------------------------------------------------------
34# 3. Create the optimizer
35# ---------------------------------------------------------------------
36optimizer = MultilayerOptimizer(
37    MS,
38    eps_model,
39    freq_optimize=np.array([8, 10, 12]),  # Optimize at 8, 10 and 12 GHz,
40    SE_min=30,
41    lb=0,
42    ub=1,
43    material_list=np.array([0, 0.125, 0.25, 0.5, 0.75, 1]),
44    n_discr=3
45)
46
47# ---------------------------------------------------------------------
48# 4. Initial guess and cost evaluation
49# ---------------------------------------------------------------------
50initial_cost = optimizer.eval_cost(x0)
51print(f"Initial cost (max R/A): {initial_cost:.4f}")
52
53# ---------------------------------------------------------------------
54# 5. Optimize multilayer parameters
55# ---------------------------------------------------------------------
56solution = optimizer.optimize(x0)
57
58# ---------------------------------------------------------------------
59# 6. Visualize the optimization results
60# ---------------------------------------------------------------------
61# Print the optimization results
62print(solution)
63
64# Visualize the optimized multilayer dielectric slab structure
65optimizer.visualize_optimization(solution)
66plt.show(block=True)
67
68# Visualize the behavior of the optimized multilayer dielectric slab structure
69freq_vec = np.linspace(8, 12, 200)  # Example frequency range [GHz]
70MS.visualize_behavior(solution, freq_vec, eps_model)
71plt.show(block=True)

Note

In this example, the discrete solution performs significantly worse than the continuous one, since it is restricted to material parameters listed in material_list. Consider adding 0.125 to the material_list and re-running the script.
