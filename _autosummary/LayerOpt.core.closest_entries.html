LayerOpt.core.closest_entries

LayerOpt.core.closest_entries(array, x, m)[source]

Find the m entries in an array closest to a given value.

This function returns the m values from array that are nearest to x. If m equals the length of array, the full array is returned. The returned values are sorted according to their original order in the array.

Parameters:

  • array (np.ndarray) – Input array to search.

  • x (float) – Target value to compare against.

  • m (int) – Number of closest entries to return. Must be less than or equal to the length of array.

Returns:

A tuple containing the m entries from array closest to x, sorted in the same order as they appear in array.

Return type:

tuple

Raises:

AssertionError – If m is greater than the length of array.
